# c_compiler

# キーワード

| No  | ワード                              | 説明                                                |
| --- | -------------------------------- | ------------------------------------------------- |
| 1   | トークン(Token)                      |                                                   |
| 2   | 構文木(Syntax Tree)                 |                                                   |
| 3   | 抽象構文木(AST: Abstract Syntax Tree) | グループ化のためのカッコなどの冗長な要素を木の中に残さずになるべくコンパクトに表現した構文木のこと |
